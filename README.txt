This project has been created as part of Machine Learning Nanodegree program of Udacity and it was also a competetion on Kaggle.
Challenge was to create models that predict the probability that a driver will initiate an auto insurance claim in the next year. Detailed information about the project can be found in the report.


Jupyter notebooks have been created using the Python 3 environment of Kernels at Kaggle.
This Python 3 environment comes with many helpful analytics libraries installed
It is defined by the kaggle/python docker image: https://github.com/kaggle/docker-python

Data folder contains csv files for training and testing that were provided in the competition.

Fo running the notebooks path for input files may have to be modified.

Submission folder contains the csv files submitted in the competition.
